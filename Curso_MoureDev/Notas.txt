Java vs JavaScript

Son Lenguajes de programación con un par de similitudes pero lo que los diferencias es que, Java es un lenguaje de programación Compilado.
JavaScript es un lenguae de programación de interpretación.

Compilado se refiere a que el codigo que se ejecute se tranformara a otra fichero hasta ser lenguaje de maquina y poder leer linea por linea
Interpretado se refiere que el codigo lo ejecuta tal cual el usuario lo escriba, no necesita de un compilado ya que lee el codigo tal cual lo esta interpretando

Otra de las grandes diferencias entre estos dos lenguajes es que JavaScript es un lenguaje con tipado debil y de tipado dinamico y Java fuertemente tipado y de tipado dinamico
El tipado (debil/fuerte/dinamico) hace referencia a indicar el tipo de dato que tendran las variable, si es de tipado debil no es necesario colocar el tipo, si es de tipado fuerte es necesario hacerlo, si es dinamico, a medida que avance el codigo puedo cambiar el tipo de dato en cualquier momento de la variable.

Buenas Practicas

Guia de estilo de JavaScript el cual contiene las buenas practicas deseables para los programadores de este lenguaje.

Kebab case : es una convención de nomenclatura en programación y desarrollo web donde las palabras se unen con guiones medios (-) y se escriben en minúsculas, como en nombre-de-archivo o mi-clase-css
snake case : convención de nomenclatura se utiliza para escribir identificadores (como variables, funciones o nombres de archivos) donde cada palabra está separada por un guion bajo (_) y todas las letras están en minúsculas [1, 2]. 

Normalmente en los ficheros de JS usan Kebab case y las snake case las usan normalmente en variables

En JS moderno, ya no es necesario colocar punto y comas

lowercamelcase: Primera palabra en minuscula y ldespues mayuscular, uso en variables
 
Interpolar una variable: es la tecnica de incorporar una variable directamente en una cadena de texto

Estructuras de datos
Existen estructuras de datos como los Arrays y los Set, son listas ordenadas pero tienen una gran diferencia la cual es que set no se maneja por indices. 
Set nos puede arrojar valores booleanos.
La principal diferencia entre los ARRAY y los SET es que los SET no admite dumplicados
La estructura map es cuando quieras utilizar los datos para modificarlos

Estructuras de control (loops o bucles) se asocian a las estructuras de datos

Objetos son una colección de propiedades, similares a los mapas a nivel estructura

Se usan las clases cuando vamos a crear multiples objetos con estructura similar y queremos reutilizar codigo
